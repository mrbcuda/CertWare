Semi-Formal Proof Review Wizard

Background

The motivation behind the semi-formal proof (SFP) argument notation is to make arguments easier to read and validate. While some of the validation is done automatically behind the scene in the argument editor, the remainder is left to the argument author or verifier. The CertWare workbench provides a "review wizard" to assist with this remaining review task.

Application

Invoke the wizard by raising the context menu (usually right-click) on any SFP file and choosing the "Review Wizard" menu item. The wizard launches in the manner of many Eclipse wizards.

The first page of the wizard prompts for validation record fields. Enter the author and time stamp values that the wizard should apply to the subsequent validation actions. The wizard loads the current time by default. For convenience the wizard also provides two buttons to clear existing validation fields if desired; simply select the button and the wizard with prompt with a confirmation before continuing to apply the change to the selected file.

The second page of the wizard presents a master-details block arrangement for navigating the argument. Line by line, select each statement in the master panel to raise its associated details in the details panel. After reviewing the statement use the validation buttons to complete the review before continuing with the next statement.

Use the Cancel button to terminate the review without saving any validation record changes. Use the Finish button to terminate the review and save any changes to the argument file.
