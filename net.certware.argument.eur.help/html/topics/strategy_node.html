Strategy Nodes

Background

A goal or set of goals (arguments in EUROCONTROL notation) can be solved by a strategy, which breaks those goals down into a number of sub-goals. The interpretation is that the solution of the sub- goals ensures the solution of the parent goals.

Application

To add a strategy node to the case argument model, right-click on an Argument node, choose Add Child... from the pop-up menu, and choose Strategy. Select the new strategy node and then complete its properties in the property sheet form.
