CertWare Examples

Identifier:
net.certware.example.net.certware.example

Since:
1.0.0

Description:
Provides case document or argument fragment examples to the run-time platform. Installs the examples as a new wizard for selection through standard workbench operations.

Configuration Markup:

<!ELEMENT extension (pattern | document | checklist)+>

<!ATTLIST extension

<!ELEMENT pattern ((related_pattern+)? | (resource+)? | (related_document+)?)?>

<!ATTLIST pattern

idp            CDATA #REQUIRED

name           CDATA #REQUIRED

author         CDATA #IMPLIED

version        CDATA #REQUIRED

intent         CDATA #IMPLIED

motivation     CDATA #IMPLIED

applicability  CDATA #REQUIRED

consequences   CDATA #IMPLIED

implementation CDATA #IMPLIED

>

  • idp - An identifier for the pattern contribution.
  • name - The name of the pattern for identification in the selection wizard.
  • author - The author of the pattern.
  • version - A version tag for the pattern definition.
  • intent - States what the pattern is trying to achieve.
  • motivation - Communicates why the pattern was constructed.
  • applicability - Records under what circumstances the pattern can and should be applied.
  • consequences - What remains to be done after applying the pattern.
  • implementation - Provides hints and techniques for successful application of the pattern.
